# Yazi Cheatsheet

## Navigation
| Key       | Action                        |
|-----------|-------------------------------|
| `h/l`     | Parent / Enter dir or open    |
| `j/k`     | Down / Up                     |
| `gg / G`  | Top / Bottom                  |
| `C-u/C-d` | Half page up / down           |
| `f<char>` | Jump to char (like vim f)     |
| `1-9`     | Relative motion (e.g. `3j`)   |
| `z`       | Zoxide jump                   |
| `C-f`     | Smart filter                  |

## Go-to Shortcuts (`g...`)
| Key  | Destination        |
|------|--------------------|
| `gh` | `~` home           |
| `gd` | `~/Downloads`      |
| `gc` | `~/.config`        |
| `gC` | `~/.config/dotfiles` |
| `gp` | `~/Pictures`       |
| `go` | `/mnt/backup/code` |
| `gt` | tig (git browser)  |

## File Operations
| Key       | Action                   |
|-----------|--------------------------|
| `Space`   | Toggle select            |
| `a`       | Create file/dir (end `/` for dir) |
| `r`       | Rename                   |
| `y / x`   | Yank / Cut               |
| `p`       | Paste                    |
| `d`       | Trash                    |
| `D`       | Delete permanently       |
| `Y`       | Copy path                |
| `C-d`     | Drag & drop (ripdrag)    |
| `C`       | Compress (ouch)          |

## Tabs
| Key       | Action         |
|-----------|----------------|
| `t`       | New tab        |
| `[  /  ]` | Prev / Next tab |
| `1-9`     | Switch to tab N|

## Bookmarks (`b...`)
| Key  | Action          |
|------|-----------------|
| `bm` | Set bookmark    |
| `b'` | Jump to bookmark|
| `bd` | Delete bookmark |

## Misc
| Key  | Action               |
|------|----------------------|
| `.`  | Toggle hidden files  |
| `!`  | Shell in CWD         |
| `~`  | Built-in help        |
| `?`  | This cheatsheet      |
